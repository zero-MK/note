ARM 只有 load/store (LDR 和 STR) 指令能直接访问内存。 x86 大部分指令都能直接操作内存中的数据，而在 ARM 上数据必须先从内存转移到寄存器中，然后才能进行操作。

## 寻址模式

- 立即数作为偏移
  - 偏移量寻址
  - 前索引寻址
  - 后索引寻址
- 寄存器作为偏移
  - 偏移寻址
  - 前索引寻址
  - 后索引寻址
- 移位寄存器作为偏移
  - 偏移寻址
  - 前索引寻址
  - 后索引寻址

例子：

```
LDR R2, [R0]   @ 从 R0 存储的地址读取数据到 R2
STR R2, [R1]   @ 把 R2 寄存器存储的数据写入 R1 存储的地址
```
